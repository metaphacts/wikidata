[[#> app:FeatureDemonstration-demosystem title="Spatial Queries & Map Component"]] [[#*inline "motivation"]]

Chemical Structure Search is used to represent chemical formulas depend on the user's request.

[[/inline]] [[#*inline "solution"]]

The user can easily search for a particular chemical formula by drawing it on the board, as well as find similar ones using similarity threshold.

[[/inline]] [[#*inline "configuration"]]

Objects within 10 km radius from the centre of Berlin


    
    PREFIX geo: <http://www.bigdata.com/rdf/geospatial#>
    PREFIX example: <http://www.example.com/>
    SELECT ?lat ?lng ?description ?link WHERE {
    SERVICE geo:search {
        ?link geo:search "inCircle" .
        ?link geo:predicate <http://www.wikidata.org/prop/direct/P625>.
        ?link geo:spatialCircleCenter "52.516666666667#13.383333333333" . # Berlin
        ?link geo:spatialCircleRadius "10" . # default unit: Kilometers
        ?link geo:locationValue ?locationValue. 
        ?link geo:coordSystem "2" . 
        ?link geo:latValue ?lat.
        ?link geo:lonValue ?lng .  
      }
      ?link rdfs:label ?description 
      FILTER (LANG(?description) = "en")
    }

Objects within a defined rectangular area


      
        PREFIX geo: <http://www.bigdata.com/rdf/geospatial#>
        SELECT ?lat ?lng ?description ?link WHERE {
          SERVICE geo:search {
                ?link geo:search "inRectangle" .
                ?link geo:predicate <http://www.wikidata.org/prop/direct/P625>.
                ?link geo:spatialRectangleSouthWest "52.5#13.35" .
                ?link geo:spatialRectangleNorthEast "52.52#13.45" .
                ?link geo:latValue ?lat.
                ?link geo:lonValue ?lng .  
                ?link geo:locationValue ?locationValue
          }
          ?link rdfs:label ?description .
          FILTER (LANG(?description) = "en")
        }
